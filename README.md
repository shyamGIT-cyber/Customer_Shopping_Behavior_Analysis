# Customer Shopping Behavior Analysis

## Overview

This project focuses on analyzing customer shopping behavior using a complete data analytics workflow. The goal of the project was to understand customer purchasing patterns, product performance, spending trends, and customer loyalty behavior through data cleaning, SQL analysis, and interactive dashboard creation.

The project includes:

* Data loading and preprocessing
* Data cleaning and transformation
* SQL-based business analysis
* Power BI dashboard development
* Business report generation
* Presentation creation using Gamma

This project helped improve practical skills in SQL, Power BI, and data storytelling while working on a real-world retail dataset.

---

## Dataset

The dataset contains customer shopping records with information related to:

* Customer demographics
* Product categories
* Purchase amounts
* Subscription status
* Shipping preferences
* Product ratings
* Discounts and previous purchases

The data was cleaned and structured before analysis to ensure accurate reporting and dashboard visualization.

## Dashboard & Analysis Highlights

The dashboard provides insights related to:

* Customer purchasing behavior
* Revenue trends
* Product category performance
* Customer segmentation
* Subscription impact on sales
* Repeat customer analysis
* Discount usage patterns
* Customer demographics and spending trends

The visualizations were designed to make business insights easy to understand and support data-driven decisions.

---

## Key Insights

* Loyal customers contributed a significant portion of total revenue.
* Subscription-based customers showed higher engagement and repeat purchases.
* Certain product categories consistently outperformed others.
* Discounts influenced customer purchasing behavior across multiple segments.
* Customer demographics played an important role in spending patterns.
